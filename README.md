# Gumatmun
Aplikasi Silsilah Keluarga Berbasis Web

# Tech Stack
- Backend: PHP 8+ with SQLite (file-based DB, no setup needed)
- Frontend: Vanilla HTML, CSS, JavaScript
- Tree Visualization: Custom D3.js-style SVG rendering for interactive family tree
- Icons: Font Awesome CDN
- Fonts: Google Fonts (Inter, Outfit)
# Fitur Lengkap
- Interactive Family Tree Visualization - Pohon keluarga interaktif dengan zoom, pan, dan klik untuk detail
- Member Management (CRUD) - Tambah, edit, hapus anggota keluarga
- Relationship Management - Hubungkan anggota (Ayah, Ibu, Pasangan)
- Member Profile - Detail profil lengkap: foto, bio, tanggal lahir/wafat, alamat, dll
- Photo Upload - Upload dan kelola foto anggota keluarga
- Search & Filter - Cari anggota berdasarkan nama, generasi, status
- Statistics Dashboard - Statistik keluarga (jumlah anggota, generasi, dll)
- Export/Print - Cetak/export pohon keluarga ke PDF/gambar
- Admin Authentication - Login admin untuk kelola data
- Responsive Design - Mobile-friendly, tablet-friendly, desktop
- Branch Family - Tambah keluarga cabang
